# F.R.A.P Charts 🏆

A beautiful ranking app where you can create custom categories and rank items within them. Track position history, add photos and comments, and see detailed statistics.

## Features

✨ **Multiple Categories** - Create as many ranking categories as you want (Movies, Albums, Games, etc.)

⬆️⬇️ **Easy Ranking** - Move items up and down to reorder your rankings

📊 **Position History** - Automatically track how long each item has been at each position

🥇 **Medal Stats** - See how long items have spent in the top 3 positions

📸 **Photos & Comments** - Add photos and notes to each item

💾 **Auto-Save** - Data is saved to your device's local storage automatically

🌙 **Beautiful Dark Theme** - Elegant UI with dark mode aesthetic

📱 **Mobile Friendly** - Fully responsive design works great on phones

## Usage

1. Open the app in your browser
2. Click on a category tab to switch between your rankings
3. Add items by typing a name and clicking "+ Adicionar"
4. Use the up/down arrows to reorder items
5. Click on any item to view details, add photos, write comments, and see statistics
6. Create new categories with "+ categoria"
